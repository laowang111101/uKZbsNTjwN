## 内容介绍

本项目是一个银行客户管理系统，主要实现银行与客户之间的基本信息管理、账户管理、交易记录管理等功能。通过本系统，可以高效地处理银行业务，提升银行的服务质量和效率。项目设计遵循实用性、安全性和可扩展性原则，为用户提供了友好的操作界面。

- **数据库：MySQL 5.7/8.0**
